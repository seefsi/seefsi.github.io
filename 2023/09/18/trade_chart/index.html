交易元素

入场点

  • 1~3次信号可以入: 弱势转强

  • 8~10次信号且波动<$2左右也可以入:

  • 错过不入:

  • 回调过半可能入:

  • 弱势机会多:

  • 回到成本区: 慢涨/反复突破时，再回到成本区，可以继续拿

  • 反弹转弱: 反弹-慢涨 -> 弱势-慢跌 -> 强势-快跌。(机会容易错过)

  • 技术机会：在M15线上找入场机会，避免随意交易。M30如果诱多，可能会砸盘。

  • 不迷惑：不要被眼前迷惑，盯盘不要看当前是涨还是跌，而是看持单人为什么进。(强者心态)

  • 加速上涨/下跌机会: 若是震荡行情，说明在搞鬼；若是单边行情，说明观望进场。

出场点

  • 位置: 逆势的开盘价、…。

  • 弱势平仓: 不划算，反弹再平

  • 状态好的时候，锻炼主动出场。(强者心态)

  • 横盘位置处于其他人的成本价，不要出。有可能还要再入场。

  • 主力持仓的目标是什么位置：震荡与下跌行情的分歧点。

  • 价格低于前高：说明开始砸了，或顶肩结构。

  • 不要把顺势单平完。

仓位

  • 网格下单: 震荡行情，(如果超过预期)白天还是一次进比较好；中长线可用网格化入场

加仓

  • 条件: M15上技术支撑，送上门的单子不要随便要。(好像均线交点当时看不出来)

止损

  • 转弱行情：快涨慢跌/快涨快跌必须出

  • 盘整突破：盘整-慢涨 -> 盘整-快跌 -> 盘整-慢涨 -> 盘整-慢跌 -> 破位。(破位跟进，不要恐惧)

  • 逆势短拿，顺势长拿：

  • 先找止损再开单：不容易被频繁打止损

  • 反向操作：止损的位置进，追高的位置出

赢亏比

  • 高赢亏比机会: 空转多，多转空 (多空互转)

空仓

  • 赢利空仓：赚够当月目标

  • 空仓等待：机会不好，行情不稳

  • 错过机会：早点选择方向，选慢了就是进入下一个横盘。错过大行情，跟风或对着干都危险。

  • 行情不好：没有1H的稳定行情，先观望。否则找入场机会。

持仓

  • 行情不好：持仓2H~4H

  • 搞鬼：低位横盘可能搞鬼，美盘要早点选择。

  • 亏损抗仓：不确定方向全部平仓。做短线久了容易对着干。

复盘

  • 复盘分析：一般是客观的

  • 大亏损：在55开的行情中，错误判断方向。盲目跟风或对着干。因此行情熟悉的越多越好。

赢利曲线

  • 大资金: 赢利曲线稳步上升，有没有跑赢大盘
