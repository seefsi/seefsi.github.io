交易元素

入场点

  • 反向操作: 再次达到入场预期，等；未不到，进

趋势入场点

  • 1~3次信号可以入: 弱势转强

  • 8~10次信号且波动<$2左右也可以入:

  • 错过不入:

  • 回调过半可能入:

震荡入场点

  • 弱势机会多:如预测到是盘整行情，那么入场点多，可以有选择合适的入场点。

  • 回到成本区: 慢涨/反复突破时，再回到成本区，可以继续拿

  • 反弹转弱: 反弹+慢涨 -> 弱势+慢跌 -> 强势+快跌。(机会容易错过)

  • 加速上涨/下跌机会: 若是震荡行情，说明在搞鬼；若是单边行情，说明观望进场。

短线入场点

  • 技术机会：弱势行情，观察M5均线转折点。

  • 技术机会：在M15线上找入场机会，避免随意交易。M30如果诱多，可能会砸盘。

  • 不迷惑：不要被眼前迷惑，盯盘不要看当前是涨还是跌，而是看持单人为什么进。(强者心态)

  • 精神角度: 分享完进场，维护完进场，休养过半就进场。

出场点

  • 位置: 逆势的开盘价、…。

  • 位置: 被套时，以之前的入场价作为出场点做波段。

  • 位置: 寻找新的多空平衡点，寻找当前背离的行情。

  • 位置: 拐点，关注各种关键点位。

  • 位置: 形态上，底部双底+回调，顶部三顶+回调。

  • 弱势平仓: 不划算，反弹再平

  • 状态好的时候，锻炼主动出场。(强者心态)

  • 横盘位置处于其他人的成本价，不要出。有可能还要再入场。

  • 主力持仓的目标是什么位置：震荡与下跌行情的分歧点。

  • 价格低于前高：说明开始砸了，或顶肩结构。

  • 不要把顺势单平完。

  • 分歧行情，出场点设在M30-2H支撑。

  • 感觉要走的单子，一定要走。

仓位

网格下单: 中长线可用网格化入场
震荡行情不可用；
早上行情，如果超过预期，还是一次进比较好；

赢利加仓：每周赢亏4:1再考虑加仓

加仓条件：M15上技术支撑
送上门的单子不要随便要。(好像均线交点当时看不出来)

长线加仓：先加后减

止赢

  • 反向操作: 达到短时的止赢预期，留；达不到，走

止损

  • 反向操作: 错过短暂的止赢，走；又来了，留

  • 转弱行情：快涨慢跌/快涨快跌必须出

  • 盘整突破：盘整-慢涨 -> 盘整-快跌 -> 盘整-慢涨 -> 盘整-慢跌 -> 破位。(破位跟进，不要恐惧)

  • 逆势短拿，顺势长拿：

  • 先找止损再开单：不容易被频繁打止损

  • 反向操作：止损的位置进，追高的位置出

  • 临近关键位置，注意 急拉急跌，止损尽量设

  • 止损要设2次，不要不设，特别是大波动时

空仓

  • 赢利空仓：赚够当月目标

  • 空仓等待：机会不好，行情不稳

  • 错过机会：早点选择方向，选慢了就是进入下一个横盘。错过大行情，跟风或对着干都危险。

  • 行情不好：没有1H的稳定行情，先观望。否则找入场机会。

  • 每月只交易3周，纪念能量不足。或者每日单子减半。

持仓

  • 行情不好：持仓2H~4H

  • 搞鬼：低位横盘可能搞鬼，美盘要早点选择。

  • 亏损抗仓：不确定方向全部平仓。做短线久了容易对着干。

赢亏比

  • 高赢亏比机会: 空转多，多转空 (多空互转)

赢利曲线

  • 大资金: 赢利曲线稳步上升，有没有跑赢大盘
